# Hello from Fromgge

We are the maker of qpRemake and qpProject, some of our projects are:

1. **qpProject** : a Fan wiki of the game Quantum Protocol, containing all game's card data and has compatibility with in-game deck import feature.
2. **qpRemake** : started out as a fan remake of the game Quantum Protocol, now an event based game engine for web-based card games.
3. **qpLang** : (Developing) a DSL for writing card effects in natural langauge
